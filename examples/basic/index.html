Vanadiel Time - Basic Example

This example demonstrates how to use the Vanadiel Time package in a simple web application.

Using the Calculator

// Import the calculator
import { VanaTimeCalculator } from 'vanatime';

// Get current Vana'diel time
const time = VanaTimeCalculator.getTime();
console.log(`Current time: ${time.formatted}`);

// Get current Vana'diel date
const date = VanaTimeCalculator.getDate();
console.log(`Current date: ${date.weekdayName}, ${date.day}/${date.month}, ${date.year}`);

// Get formatted date and time
const formatted = VanaTimeCalculator.formatDateTime();
console.log(`Formatted: ${formatted}`);

Using the Components

In a Vue 3 application:

// In your Vue component
import { SimpleVanadielClock, VanadielTime } from 'vanatime';

export default {
  components: {
    SimpleVanadielClock,
    VanadielTime
  }
}

In your template:

<template>
  <SimpleVanadielClock />
  <VanadielTime @update="handleTimeUpdate" />
</template>
